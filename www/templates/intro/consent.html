Traceur de Mobilité / Fabrique des Mobilités / VERSION TEST PHASE 2

Context and goals

The app Traceur de Mobilité FabMob is a mobile application reusing the e-mission software developed at Berkeley University by Shankari K and published as an open source software (https://github.com/emission).
This project of "la Fabrique des Mobilités" is currently (Summer 2019) in an evaluation phase (internal test, on invitation), the application is not yet public.

The current version of the app is A TEST VERSION. The goal is of the test phase is to invite interested actors to evaluate the application, and the relevance of this software for meeting the use cases that interest them.
For this test phase, no particular level of service shall be expected from the server.
When creating an account in order to use this application, the user acknowledges to have read and understood the project goals and has to accept the terms of use before starting to use it.

Use of collected data

We automatically track your location, accelerometer, device-generated activity recognition, and battery usage. We also allow you to optionally enter your experiences and feedback during travel through the app, including answering to external surveys, via the LimeSurvey service which we host on our server. Since our code is open source, you can inspect the data that we collect in the background. - the users can download their personal data and ask that their account and data be deleted at any time
- the users can see and inspect their data via the app
- in a few weeks, the user will also be able to view his/her data via web browser (grfmap application)
- the project team (administrators) can see all data (data are not encrypted)
- the data are only used for, they are not transmitted to any third part, nor are they used for any study
- the data will be deleted when 120 days maximum after the beginning of the evaluation phase
